# azure-devops-mcp-skills

Skills para facilitar o uso das tools do MCP Azure DevOps no OpenCode.

## O que e / quando usar

Este projeto reune skills que orientam o OpenCode a executar operacoes comuns no Azure DevOps via MCP com mais clareza e previsibilidade. Use quando voce precisa automatizar ou padronizar a interacao com itens de trabalho, repositorios, pipelines e outros recursos. Nao substitui a documentacao oficial; serve como guia pratico para chamadas recorrentes.

## Principais dominios cobertos

- itens de trabalho
- repositorios
- pipelines
- wiki
- planos de teste

## Limitacoes / nao e

- depende do MCP Azure DevOps configurado no OpenCode
- nao cobre 100% das operacoes do Azure DevOps

## Status do projeto

Em desenvolvimento (colecao em expansao).
